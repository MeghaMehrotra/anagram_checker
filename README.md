# Anagram Checker

This program checks whether the two given strings are anagrams or not.

### Prerequisites

* JDK 8
* Apache Maven

### How to build & test

For Building the project:
Go to the project path ./anagram_checker and type the below command
```bash
mvn clean install
```

For testing:
Go to the project path ./anagram_checker and type the below command

```bash
mvn test
```
